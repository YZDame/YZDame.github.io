如何当好一名校内竞赛教练

发表于 2025-12-22 更新于 2026-03-06 分类于 竞赛方法论 本文字数： 3.6k 阅读时长 ≈ 6 分钟

一、序言：校内竞赛教练vs机构老师

校内教练多了哪些职责？

机构老师：

  • 几乎只需完成教学任务

  • 教学任务设计相对随心所欲

  • 需要承担少部分招生任务

校内教练的额外压力：

1. 教学上的"委屈"

  • 兼顾课内外教学：很多老师要同时负责高考、竞赛、强基三块，备课时间大大增加

  • 生源基础薄弱：在弱校中，很难招到初中基础不错的学生，每届可能只有一两个，教学节奏必须兼顾课内进度

  • 学生水平参差不齐：不像机构可以把水平接近的孩子凑到一起，竞赛组水平天南海北，教学难以照顾所有阶层

  • 被低估的水平：人们总觉得机构老师水平更高，这显然不公平

2. 教学内容的内在枷锁

  • 承担铺基本方法的任务：比起讲新奇问题，校内教练要更多地承担把普通方法技巧给学生铺一遍的任务

  • 两类问题的矛盾：

  • 学生听了会觉得老师水平很高的问题：有丰富背景的问题、高级定理、巧妙方法、大学教材/论文问题、游戏策略类问题

  • 优秀学生会听起来不满意的问题：基础方法讲解、一试内容、老题、不美观的问题

关键矛盾：机构老师可以完全避免第二类问题，而校内老师几乎不可避免要讲大量第二类问题。如果是成名教练，学生可以理解；但如果是非清北出身的年轻老师，很容易被学生看不起。

3. 非教学事务

  • 学习相关：跟进学生学习情况、检查听课质量、指导自习、安排模拟测试

  • 非学习相关：竞赛生的"超级班主任"，负责竞赛生的每分每秒、招生、德育、甚至研究高等知识

  • 最终问题：学校没钱，做这些干啥？（建议直接跳槽）

二、学生的学习安排

竞赛生的发展周期

  1. 预备期：预习初高中课内知识，达到高考130分左右水平

  2. 入门期：开始二试学习，四大板块过完一遍，课外以看书和一轮课程为主

  3. 联赛刷题期：学习二试二轮，刷近年赛题，了解CMO热门知识点，参加机构外培

  4. CMO刷题期：准备CMO，针对训练CMO3、6，周期性练习联赛一试保持手感，参加更难外培

预备期

  • 推荐资料：《六十课时学完高中数学》

  • 尽量少在初联和小奥上花费时间

  • 有条件的话，直接尽快进入二试学习

入门期（一二试一轮学习）

听什么课

  • 外部课程：陈祖维、杨全会的线上课程（极好，必听）

  • 自己授课：承担一试、几何、数论梳理任务（根据自身水平和时间，排名分先后）

  • 代数：想讲好代数对教练要求极高，需要很强的命题风向把控能力，推荐学习龚固老师公众号《天空的数学小天地》

看什么书

一试：浙大和中科大的几本书仍是一轮学习的优质选择

几何：

  • 入门：小蓝皮几何

  • 主要：浙大《几何分册》和Evan Chen《数学奥林匹克中的欧几里得几何》

  • 理想组合：后者 + 前者关键章节（圆与圆相交、圆与圆相切、直角三角形）

  • 辅助：《几何变换与几何证题》、《命题人讲座圆》、万喜人

代数：

  • 入门：小蓝皮4、9两本

  • 系统性方法：目前书籍几乎无法提供帮助

  • n元不等式理论：《代数不等式的证明》、《不等式的秘密》第一卷、《上海中学竞赛课程》

  • 关键：当前代数真的很靠网课，需要优秀老师总结近年赛题，提点新式赛题思路

数论：

  • 神书：《命题人讲座初等数论》和Titu《数论：概念与问题》

  • 建议：小蓝皮数论打底，然后以学习上面两本书之一为主

  • 校内教学：直接选一本作为蓝本，需要对知识体系做一定补充或增删

组合：

  • 入门：小蓝皮两本组合

  • 思路讲解：冯跃峰《组合构造》（有些过时但思路好）

  • 全面性：《奥赛经典组合分册》（堪堪能看）

  • 综合最好：《上海中学数学课程》（题目新颖，讲解不错，但初学者难看进去）

  • 推荐：两本小蓝皮打底，同步学习《组合构造》、《上中》，刷俄罗斯赛题

联赛刷题期

停不停课与竞赛需要多少时间

  • 不停课：保证周一到五每天2-3小时竞赛学习，周末几乎全空闲即可

  • 停课：更好，但必须有人盯着一举一动

  • 关键：一天能集中注意力搞数学的时间约4-5小时，再多效率下降

做什么题：一试

  • 可做：中数、根源杯、学数学

  • 避雷：协作体（偏题、老题、错题多）

  • 推荐：数之谜"筑梦"模拟（质量极高）

做什么题：二试原则

  1. 年份考察：远于十年的题尽量不选（除重要考试或特别经典）

  2. 优先刷完中美俄大赛题（可刷多遍）

  3. 有阶段性主题（板块或考试）

  4. 引导学生情绪，忌讳评价题目好坏

做什么题：题源顺序

  1. 东南、西部、女奥、高联、CMO

  2. IMO预选题、IMO、08~14CTST

  3. 中美俄其它比赛（TST,TSTST,ELMO等）

  4. 日本、越南、罗马尼亚、保加利亚等国的MO,TST

  5. 其余国家考试、网上原创题

品味与趋势

  • 品味定义：近五年所有选拔考试赛题风格的平均值

  • 联赛趋势：没有趋势，不想被预测

  • 避免：过于老的问题（如几何老旧三角形性质题、代数传统不等式）

建议着重练习的板块

  • 几何：自由度≥4的问题（四边形出发）、动点问题

  • 代数：非常规代数问题（ISL数列、离散极值、代数杂题、形式新颖不等式）

  • 数论：数论与数列、组合结合的问题，有一定高等背景的问题

  • 组合：与其余板块结合、游戏策略类、分析特殊结构、崭新情境

可以减少练习的板块

  • 几何：三角形五心性质/经典性质出发的问题

  • 代数：三元不等式、除均值和柯西以外定理结论为核心的不等式

  • 数论：不定方程、结论导向型问题

  • 组合：结论导向型问题

一篇重要的阅读理解（关于CMO命题趋势）

问题背景：考生批评连续三年CMO集训队线87分加权，金牌线水涨船高

重要观点：

  1. 变卷是我国学生人数众多和优质资源稀缺导致的必然结果

  2. 命题组明确希望在1、2、4、5题减少区分度，防止只训练这些的学生进集训队

  3. 命题组在3、6题设置难度较低的得分点

重要提示：联赛压力较小的同学，平时训练不能排斥3、6难度的顶级难题，多进行讨论，即使无法完整做出，讨论产生的部分思路可能转化为最后的部分分。

模拟测试

  • 质量通常不如赛题

  • 用途：训练后期、需要考试排名选拔时

  • 推荐：大机构联考题、易湃刷题班

上什么课

外培目的明确：

  1. 一定要上命题人的课

  2. 注意外培节奏，劳逸结合，与日常训练穿插

学生的非授课指导

指导方式

  1. 督促学生做难题集

  • 金牌选手至少十本起步

  • 从被动监督到主动热情

  • 记下思路与标答/老师思路"打架"的过程

  1. 面批

  • 叫1-3名学生到办公室

  • 检查学习内容、上课笔记

  • 让学生给自己讲一遍或直接在边上写

  • 注意：不管学生水平多高，都可以严厉严格进行

  1. 做过的题重做

  • 从作业/考试中选题，尤其是外培题

  • 严谨复写是初学和独立解题之间的良好过渡

  • 锻炼书写

  1. 讨论班/学生讲题
  • 准备问题或知识点，让学生自学后互相讲解

指导要点：及时敲打不当心态

常见不当心态：

  1. “xxx太牛了”、"mol"等起哄

  2. 对题目挑三拣四（太偏、出题人脑残）

  3. 畏新、畏难

  4. 对平面几何、对称不等式有过大热情

  5. 做题好高骛远

  6. 对网络活动、原创题过于热情

敲打方式：多用案例提点，必要时请老学长回来讲讲

指导要点：“过拟合”

定义：机器学习术语，指模型只适用于特定数据集

学生表现：

  1. 对冷门/高级方法非常推崇，做题想不到或过早放弃基本方法

  2. 希望建立绝对映射，把所有问题题干对应到唯一方法，不允许通过巧思或尝试解决

预防：在种子选手身上多加预防

指导要点：德育

  • 数之谜联赛考前考后情绪反差大

  • 原因：一半好高骛远，一半弄虚作假，仅10%是真发挥失常

  • 核心：保证学生始终在向前做功

三、学生的挑选与选拔

招生策略

  • 教小奥课去挖人

  • 和机构喝酒、混家长圈子

  • 向校长要政策：免中考、保班型

选拔技巧

核心：不要选拔超前学的孩子，要选拔理解能力强的孩子

方法：现学现考

  • 讲偏序集和全序集定义，让学生判断、举例

  • 问"一个偏序集里有没有可能恰有0、1、3个极大元"等问题

  • 对初中生思维的很好考验

  • 小学生可用三角函数基本知识

学生家长常见负面观点及应对

  1. 竞赛后面学不懂怎么办 → 一试也对强基有用

  2. 竞赛太卷 → 咱在弱省，进省队容易，银牌也能华五强基

  3. 强基只能读理科，没前途 → 强基也有越来越多工科院系，数学系就业极好

  4. 学校前几年成绩不好 → 今年加大投入，校长开会说了

维持适当人数

以年均5省队、有初中部的学校为例：

  • 初一下学期：开始数竞选拔和一试培训

  • 竞赛组人数参考：初二50人 → 初三50人 → 高一30人 → 高二15人

两个生活小问题

在了解Frobenius问题之前, 我们先来看两个生活中的小问题.

首先, 为什么高铁的座位设置大多是 “2+3” 的组合？即过道一侧是两个座位组合在一起、另一侧是三个座位组合在一起呢？设想一下, 当你和家人或是朋友出行时, 不管你们人数是多少, 只要不少于两人, 总有一种座位组合方式让每一个人和认识的人坐在一起, 而不会出现落单的情况. 当出行人数是偶数时, 只需都安排两人坐即可；当出行人数是奇数时, 只需安排三个人坐三人座, 剩下的人（如果有的话）一定是偶数, 再安排两人座若干即可.

2024年阅读摘录

发表于 2024-10-11 更新于 2026-03-06 本文字数： 838 阅读时长 ≈ 2 分钟

余秋雨《文化苦旅》

只有走在路上，才能摆脱局限，摆脱执着，让所有的选择、探寻、猜测、想象都生机勃勃。

吴军《数学之美》

每当弗莱德和我谈起各自少年时的教育，我们都同意这样几个观点。首先，小学生和中学生其实没有必要花那么多时间读书，而他们的社会经验、生活能力以及在那时树立起的志向将帮助他们的一生。第二，中学阶段花很多时间比同伴多读的课程，上大学以后用很短时间就能读完，因为在大学阶段，人的理解力要强得多。举个例子，在中学需要花500小时才能学会的内容，在大学可能花100小时就够了。因此，一个学生在中小学阶段建立的那一点点优势在大学很快就会丧失殆尽。第三，学习（和教育）是持续一辈子的过程，很多中学成绩优异的亚裔学生进入名校后表现明显不如那些出于兴趣而读书的美国同伴，因前者持续学习的动力不足。第四，书本的内容可以早学，也可以晚学，但是错过了成长阶段却是无法补回来的。（因此，少年班的做法不足取。）现在中国的好学校里，恐怕百分之九十九的孩子在读书上花的时间都比我当时要多，更比贾里尼克要多得多，但是这些孩子今天可能有百分之九十九在学术上的建树不如我，更不如贾里尼克。这实在是教育的误区。

丘成桐《我的几何人生》

从这次和多年来同行间的摩擦，可见世事从来就不是一帆风顺的，就算拿到菲尔兹奖，人生也不会就此步步高升。地心吸力会发挥作用，拖你后腿，有时甚至拖垮你。

虽然金钱需要考虑，但正如五柳先生说的，它并不是一切，做人要看远些。我会在高研院的一年尽力工作和学习，然后再找更理想的位置。

自1987年开始，过去三十多年我都在哈佛度过。诚然，这么多年，不一定天天都快乐，但总的来说是美好的日子，或者谚语说得对，“第三次，便顺意”。

顾险峰

大多数浅层次的智力劳动很快会被AI所承担，年轻人求学期间尽量学习更为深刻的理工科方向，才能保证长期的核心竞争力。

沃尔特·艾萨克森 《埃隆·马斯克传》

先设定一个不切实际的时间表，然后“取法乎上得其中”——疯狂的想法原本不可能变成现实，结果是很多年之后最终实现了。
